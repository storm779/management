Migration Instructions
CSV Format Expected:

MSGID,MESSAGE,VALIDUPTO,PRIORITY,MSGBY,DTSTAMP,MSG_HEADER,VALIDFROM,ENABLED,MESSAGE_HINDI,MSG_HEADER_HINDI

Field Mapping:
  • MSGID → Auto-generated ID (ignored)
  • MESSAGE → Message content (required)
  • VALIDUPTO → Valid to date (optional)
  • PRIORITY → Message priority (1-10)
  • MSGBY → Created by user ID
  • DTSTAMP → Original timestamp
  • MSG_HEADER → Message header (required)
  • VALIDFROM → Valid from date (required)
  • ENABLED → Y/N for enabled status
  • MESSAGE_HINDI → Hindi message (optional)
  • MSG_HEADER_HINDI → Hindi header (optional)
Date Format:

Dates should be in M/d/yy format (e.g., 12/15/16)
